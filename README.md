# student_grade_tracker
student grade tracker.Build a Java program to input and manage student grades.Calculate average, highest, and lowest scores. Use arrays or ArrayLists to store and manage data. Display a summary report of all students ·Make the interface GUI-based as desired.
Features implemented by role:
Admin gets full access — dashboard with class-wide stats, manage students (add/edit), view all teachers, subjects overview, enter/edit/delete any grades, attendance tracking, generate report cards for any student, and system settings.
Teacher can enter and update marks by student/subject, track attendance, and generate report cards.
Student sees their personal dashboard with GPA/CGPA, subject-wise grade bars, analytics with personalized study suggestions, attendance status (flagged if below 75%), and their full printable report card.
